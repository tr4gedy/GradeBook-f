**Техническое задание: Журнал успеваемости**
---
**1. Описание проекта**
Данный проект представляет собой разработку программной системы для журнала умпеваемости. Система позволяет просматривать группы и студентов, выдавать студентам оценки, редактировать и просматривать оценки, формировать ведомость, экспортировать в пдф, а также рисовать гистограмму на основе оценок студентов.

Тип работы: учебная практика

---
**2. Цель и задачи проекта**
Цель: разработка системы журнала успеваемости.

Задачи:

анализ предметной области
разработка архитектуры системы
реализация программного решения
тестирование и оценка эффективности
Область применения: пользовательское

---
**3. Функциональность**
мониторинг доступных самокатов
начать поездку
закончить поездку
история поездок
Ограничения:

тестовый режим эксплуатации
**4. Архитектура проекта**
Тип архитектуры: клиент-сервер

Технологии:

С#
Visual Studio Community
postgresql
Структура проекта:

prct/ — исходный код
prct/Data — серверная часть
prct/view — клиентская часть
GradeBook.test/ — тесты
5. Установка и запуск
Требования:

.NET 8
WPF
Docker (опционально)
Установка:

git clone https://github.com/tr4gedy/GradeBook-f
cd GradeBook-f
Запуск:

dotnet run

---
**6. Использование**
После запуска системы пользователь может:

просматривать данные
выдавать и редактировать оценки
экспортировать ведомость в пдф
просматривать гистограмму
